# Design Context

งานที่แตะ UI/ดีไซน์ให้อ่าน 2 ไฟล์นี้ก่อน (สร้างโดย impeccable):

- **[PRODUCT.md](PRODUCT.md)** — register=`product`, ผู้ใช้/จุดประสงค์, brand personality (มั่นใจ–น่าเชื่อถือ / decision-grade), anti-references (เลี่ยงทั้ง generic SaaS dashboard, พอร์ทัลราชการรก, แอปคอนซูเมอร์เล่น ๆ, enterprise หนัก ๆ), 5 design principles, accessibility = WCAG AA + ตัวอักษรใหญ่/อ่านง่าย
- **DESIGN.md** — visual system จริง (สี/ฟอนต์/คอมโพเนนต์) ตาม Google Stitch format

Brand CI: 3 สีเท่านั้น (`#ff5008` accent only / `#000000` / `#ffffff`) · IBM Plex Sans Thai · UI ภาษาไทย
Impeccable absolute bans (ห้ามเด็ดขาด): side-stripe border >1px, gradient text, glassmorphism เป็น default, hero-metric template, การ์ดไอคอนเหมือนกันเรียงพรืด, eyebrow uppercase เหนือทุก section
